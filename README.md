# Customer-Classification-Using-Artificial-Neural-Networks-PyTorch-
This project implements an Artificial Neural Network (ANN) using PyTorch for a classification problem.The model is trained on preprocessed data and evaluated using training and validation loss.

Features:
  Data preprocessing and train-test split
  Conversion of data into PyTorch tensors
  Custom ANN architecture
  Training using backpropagation and Adam optimizer
  Evaluation using validation loss
  Loss visualization during training
  Technologies Used
  Python
  PyTorch
  NumPy
  Pandas
  Scikit-learn
  Matplotlib
Model Architecture:
Input Layer: Based on the number of input features
Hidden Layers: Fully connected layers with activation functions
Output Layer: Number of neurons equal to the number of classes
